REST API

Interactive SwaggerUI documentation can be found at /api-docs.
To create users, either use /api-docs or Postman (or similar) and POST the sample data.
Launch Swagger UI
REST
                  
Method: POST http://localhost:3000/user
Headers:
  Content-Type application/json
  Authorization accessToken_from_login
Body: raw
{
  "email": "vader@bronies.com",
  "name": {
    "first": "Darth",
    "middle": "Mortimer",
    "last": "Vader"
  },
  "role": "Manager",
  "userStatus": true,
}

GraphQL

Interactive GraphQL Explorer can be found at /graphql.
To create users, use the Explorer to run the sample query.
Launch Explorer
GraphQL

# Operation
mutation CreateUser($userInput: UserInput!) {
  createUser(userInput: $userInput) {
    id
  }
}

# Variables
{
  "userInput": {
    "email": "vader@bronies.com",
    "name": {
      "first": "Darth",
      "middle": "Mortimer",
      "last": "Vader"
    },
    "role": "Manager",
